## 🚀 How to Use

### **Step 1: Setup**
1. Enter client name (e.g., "Vive Orgánico")
2. Select month and year
3. Add objective and target audience
4. Click "Crear Presentación"

### **Step 2: Add Posts**
1. Click "+ Agregar Post" in the sidebar
2. Fill in:
   - Platform (Instagram, Facebook, TikTok, YouTube)
   - Post type (Post 4:5, Reel, Story, Video, Carousel)
   - Date
   - Content pillar (Lifestyle, Efemérides, Comunidad, Productos)
   - Visual suggestions
   - Copy In (title/hook)
   - Copy Out (main caption)
   - Hashtags
3. Click "Agregar Post"

### **Step 3: Navigate Slides**
- Use sidebar to jump to any slide
- Use "Anterior/Siguiente" buttons to navigate
- Edit or delete posts from their slides

### **Step 4: Export to PDF**
- Click "📄 Exportar PDF"
- Browser print dialog opens
- Save as PDF
- Share with clients!
