Procesar un Pedido Nuevo

1.1 — Ver Pedidos Pendientes

Navega a: Seller Center → Orders → Manage Orders

Filtra por estado:

Awaiting Shipment — Pedidos pagados, pendientes de enviar
Awaiting Collection — Ya tienen label, esperando recogida del carrier
In Transit — Ya se envió, en camino al cliente
Delivered — Entregado exitosamente
Cancelled / Return — Cancelado o en proceso de devolución

1.2 — Confirmar y Preparar Pedido

  1. Clic en el pedido con estado "Awaiting Shipment"
  2. Verifica los detalles: producto, variante, cantidad, dirección de envío
  3. Verifica que tienes stock del producto
  4. Si usas TikTok Shipping: clic en Arrange Shipment
  5. Si usas Seller Shipping: selecciona tu carrier e ingresa el tracking number manualmente
⚠️ TIEMPOS CRÍTICOS

Debes procesar el pedido dentro de las primeras 24-48 horas. Retrasos frecuentes bajan tu SPS (Shop Performance Score) y pueden limitar visibilidad de tu tienda.

2

Imprimir Etiquetas de Envío

2.1 — Con TikTok Shipping

  1. Después de "Arrange Shipment", TikTok genera la etiqueta automáticamente
  2. Ve a Orders → Manage Orders → selecciona pedido(s)
  3. Clic en Print Shipping Label
  4. Puedes seleccionar múltiples pedidos y hacer batch print
  5. Imprime en papel adhesivo 4×6" (estándar de shipping label)
  6. Pega en el paquete y lleva al punto de drop-off del carrier o agenda pickup

2.2 — Con Seller Shipping (Self-Ship)

  1. Genera la etiqueta desde tu cuenta del carrier (UPS, FedEx, USPS)
  2. Copia el tracking number
  3. Vuelve al pedido en Seller Center → clic en Ship Order
  4. Selecciona el carrier y pega el tracking number
  5. Clic en Confirm

El tracking DEBE ser válido. TikTok verifica automáticamente que el número sea real y que tenga movimiento.

💡 PRO TIP — BATCH PROCESSING

Procesa pedidos en lotes 2-3 veces al día (mañana, mediodía, tarde). Esto es más eficiente que uno por uno y te ayuda a cumplir los tiempos de envío consistentemente.

3

Gestionar Devoluciones y Reembolsos

3.1 — Tipos de Solicitudes

Navega a: Seller Center → Orders → Returns & Refunds

💰 Refund Only (sin devolver producto)
Cliente pide reembolso sin enviar de vuelta. Típico para productos dañados de bajo valor o errores de envío.
📦 Return & Refund (devuelve y reembolsa)
Cliente devuelve el producto y recibe reembolso. Es el caso más común.

3.2 — Flujo de una Devolución

  1. Cliente solicita return — Aparece en Returns & Refunds como "Pending"
  2. Revisar la solicitud — Ver motivo, fotos del cliente (si las incluye), y decidir:
    Approve o Reject
  3. Si apruebas: TikTok genera etiqueta de retorno para el cliente
  4. Cliente envía de vuelta — Recibes el paquete en tu Return Address
  5. Confirmar recepción — Una vez recibido y verificado, confirma en Seller Center
  6. Reembolso se procesa — TikTok reembolsa al cliente automáticamente

3.3 — Tiempos Límite para Responder

4

Cancelaciones de Pedidos

4.1 — Cancelación por el Cliente

El cliente puede cancelar si el pedido aún no se ha enviado.

  • Aparece en Orders → Cancellation
  • Si no has procesado el envío → aprueba la cancelación
  • Si ya imprimiste label pero no enviaste → cancela el label y aprueba
  • Si ya enviaste → rechaza cancelación, el cliente debe esperar y hacer return

4.2 — Cancelación por el Seller (Nosotros)

Solo cancelar como seller en casos extremos:

  • Sin stock — Producto agotado (debe ser excepcional)
  • Dirección inválida — Dirección del cliente no existe o es incompleta
  • Fraude detectado — Orden sospechosa
⚠️ IMPACTO DE CANCELACIONES

Cada cancelación por parte del seller afecta negativamente tu SPS. Alta tasa de cancelación puede resultar en restricciones de cuenta. Mantén tu inventario actualizado para evitar esto.

Fulfilled by TikTok (FBT)

5.1 — ¿Qué es FBT?

FBT es el equivalente de Amazon FBA para TikTok Shop. Tú envías tu inventario a los warehouses de TikTok y ellos se encargan de:

📦
Almacenamiento
90 días gratis
🎁
Empaque
Pick, pack & ship
🔄
Devoluciones
Manejo completo

5.2 — Cómo Activar FBT

Navega a: Seller Center → Logistics → Fulfilled by TikTok

  1. Verifica si tienes invitación (basado en volumen de ventas)
  2. Acepta los términos de FBT
  3. Selecciona productos para enviar a warehouse de TikTok
  4. Crea un Inbound Shipment (envío de inventario al warehouse)
  5. Imprime labels de FBT para tus cajas
  6. Envía el inventario al warehouse asignado
  7. TikTok recibe, escanea y activa tu inventario

Una vez activo, los pedidos de esos productos se procesan automáticamente por TikTok. No necesitas hacer nada.

💡 BENEFICIOS DE FBT
  • Entrega más rápida → mejor conversión
  • Badge de "Fulfilled by TikTok" en listings → mayor confianza
  • Protección contra claims de envío
  • Menos trabajo operativo diario

Tiempos de Envío y Penalizaciones

6.2 — Cómo Proteger tu SPS

  • Procesa pedidos mínimo 2x al día (mañana y tarde)
  • Mantén inventario actualizado — Sin stock = cancelaciones = SPS baja
  • Usa TikTok Shipping para trackeo automático y tiempos garantizados
  • Responde solicitudes de return en < 24h (no esperar las 48h)
  • Responde mensajes de clientes en < 12h
  • Minimiza la tasa de cancelación por debajo del 2%
  • Monitorea tu SPS semanal en Performance → Shop Health

Late Dispatch Rate (LDR) — Enforcement Activo

7.1 — Qué Cambió

Desde el 6 de abril de 2026, TikTok Shop reactivó el enforcement del Late Dispatch Rate (LDR). Esto significa que las órdenes despachadas tarde ya vuelven a generar acciones correctivas automáticas sobre la cuenta, no solo advertencias.

El LDR mide el porcentaje de pedidos que se despachan fuera del tiempo permitido. Es una de las métricas que más rápido deteriora la salud de la tienda si se descuida.

7.2 — La Regla del Tiempo de Despacho

Tienes 2 días hábiles para despachar desde que la orden entra al estado "Awaiting Shipment". Si el ship-out ocurre después de esa ventana, el pedido cuenta como despacho tardío y suma al LDR.

7.3 — Monitoreo Diario (Responsabilidad del Equipo)

El LDR es una métrica diaria: el equipo debe monitorearla todos los días desde el módulo de pedidos, no revisarla solo una vez por semana.

  • Revisa Orders → Manage Orders y prioriza los pedidos más antiguos en "Awaiting Shipment"
  • Cualquier pedido que se acerque a las 2 días hábiles pasa a ser prioridad máxima del día
  • Si el LDR empieza a subir, despacha en batch de inmediato antes de cualquier otra tarea operativa
⚠️ REGLA DEL 10%

Un LDR superior al 10% dispara acciones correctivas sobre la cuenta. Despacha siempre dentro de los 2 días hábiles desde "Awaiting Shipment" y revisa la métrica todos los días en el módulo de pedidos.

Partial Refund — Herramienta para Proteger el SPS

8.1 — Qué Es

Desde marzo de 2026, los sellers pueden ofrecer un reembolso parcial en lugar de una devolución completa para resolver issues menores. En vez de aprobar un return completo, le devuelves al comprador una parte del valor y este se queda con el producto.

8.2 — Cómo se Configura

El partial refund se configura con reglas por:

🗂️ Categoría
Defines a qué categorías de producto aplica la regla.
💵 Precio del item
Rango de $10 a $100 de valor del producto.
🏷️ Razón de devolución
Aplica según el motivo que reporta el comprador.
📊 Porcentaje
Reembolso de 10% a 50% del valor del item.

8.3 — Por Qué Protege el SPS

Lo clave: si el comprador acepta el partial refund, el caso NO cuenta contra el Seller-Fault Return Rate. Como esa métrica alimenta el Shop Performance Score (SPS), resolver por reembolso parcial evita el golpe que sí daría un return completo por falla del seller.

  1. El comprador abre una solicitud por un issue menor
  2. Le ofreces un partial refund en lugar de aprobar el return completo
  3. Si lo acepta → se queda con el producto y recibe el % acordado
  4. El caso no suma al Seller-Fault Return Rate → el SPS queda protegido

8.4 — Cuándo Conviene Usarlo

  • Issues menores donde el producto sigue siendo usable (defecto cosmético leve, empaque dañado, un detalle del que el comprador se queja)
  • Productos de valor bajo a medio ($10–$100), donde el costo y la logística de un return completo no compensan
  • Cuando el reembolso del retorno y reenvío saldría más caro que ceder un 10%–50% del item
  • Cuando quieres cerrar rápido el caso y mantener la satisfacción del comprador sin sumar al Seller-Fault Return Rate

No lo uses cuando el producto llegó completamente inservible o cuando el comprador exige explícitamente la devolución total: ahí el partial refund no resuelve y solo retrasa.

Regla de Oro

Para issues menores en productos de $10–$100, ofrece primero un partial refund (10%–50%). Si el comprador lo acepta, el caso no afecta el Seller-Fault Return Rate y proteges el Shop Performance Score.
