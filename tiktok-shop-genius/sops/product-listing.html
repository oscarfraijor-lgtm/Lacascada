Agregar y Optimizar Productos

Básico → Avanzado ⏱ 12-20 min 📅 Actualizado: Marzo 2026

Guía completa para subir productos, optimizar listings para conversión, carga masiva, requisitos de imágenes y SEO dentro de TikTok Shop.

1

Agregar un Producto Nuevo (Paso a Paso)

1.1 — Ir a la Sección de Productos

Navega a: Seller Center → Products → Add Product

También puedes usar el atajo + Add Product en la barra superior.

TikTok te ofrece dos opciones:

✅ Quick Listing
Pega un link de Amazon/Shopify y TikTok importa los datos. Luego editas.
📝 Manual Listing
Llenas todo desde cero. Más control sobre cada campo.

1.2 — Seleccionar Categoría del Producto

Este es el paso más importante. La categoría determina:

  • Qué atributos debes llenar (ingredientes, materiales, etc.)
  • La comisión que TikTok cobra por venta
  • En qué búsquedas aparece tu producto
  • Si necesitas certificaciones adicionales

Usa la barra de búsqueda para encontrar la categoría más específica posible. Ejemplo: no uses "Beauty" genérico → usa "Skincare → Face Moisturizers".

1.3 — Escribir el Título del Producto

El título debe tener entre 25 y 80 caracteres. Fórmula recomendada:

[Marca] + [Producto] + [Beneficio/Atributo Clave] + [Variante]
✅ "CeraVe Moisturizing Cream | Dry to Normal Skin | 16oz"
✅ "LED Ring Light 10" with Tripod Stand | USB Powered | 3 Light Modes"
❌ "Amazing cream best product ever BUY NOW!!!"

1.4 — Escribir la Descripción

La descripción debe ser informativa y persuasiva. Estructura recomendada:

Párrafo 1 (Hook): ¿Qué problema resuelve? ¿Por qué lo necesitan?
Párrafo 2 (Features): Lista de características principales con bullets
Párrafo 3 (Specs): Especificaciones técnicas (tamaño, peso, materiales)
Párrafo 4 (Incluye): ¿Qué viene en la caja?

Keywords: Incluye términos de búsqueda naturales. Si vendes un protector de pantalla, incluye "screen protector", "tempered glass", "phone protection".

1.5 — Configurar Precio e Inventario

Price → Precio de venta al público (incluye tu margen + comisión TikTok)
Compare at Price → Precio anterior (para mostrar descuento tachado)
SKU → Código interno del producto
Stock / Quantity → Cantidad disponible (actualizar constantemente)
💡 PRO TIP — PRICING

Recuerda que TikTok cobra ~6% comisión + 2.24% transaction fee. Tu precio debe absorber esto y mantener margen. Fórmula: Costo × 2.5 a 3x mínimo para productos con affiliate commissions.

1.6 — Configurar Peso y Dimensiones de Envío

Package Weight → Peso del paquete listo para enviar (lbs o kg)
Package Dimensions → Largo × Ancho × Alto del paquete

Estos datos afectan directamente el costo de envío calculado. Sé preciso para evitar cargos adicionales.

1.7 — Publicar y Esperar Revisión

Clic en Submit para enviar el producto a revisión.

  • Revisión automática (AI) + manual: 24-48 horas
  • Categorías restringidas pueden tomar más
  • Si se rechaza, verás el motivo en Products → Quality Issues
  • Corrige y reenvía. No crees un listing duplicado.

Requisitos de Imágenes y Videos

2.1 — Especificaciones de Imágenes

Mínimo
3 imágenes
Recomendado
8+ imágenes
Resolución mínima: 600×600px | Ideal: 1200×1200px
Formato: JPG, PNG (max 5MB por imagen)
Ratio: 1:1 (cuadrado) obligatorio para la imagen principal
Fondo: Blanco o limpio para imagen principal

Orden recomendado de imágenes:

  1. Hero shot — Producto sobre fondo blanco, vista frontal
  2. Ángulo alternativo — Vista lateral o trasera
  3. Detalle/close-up — Textura, etiqueta, componentes
  4. Lifestyle — Producto en uso / contexto real
  5. Escala/tamaño — Producto junto a referencia de tamaño
  6. Packaging — Cómo llega al cliente
  7. Infográfico — Beneficios o features en imagen
  8. Before/After o Variantes — Si aplica

2.2 — Video de Producto

Los listings con video convierten significativamente mejor.

Duración: 15-60 segundos ideal
Formato: MP4 (max 20MB)
Resolución: 720p mínimo, 1080p ideal
Ratio: 9:16 (vertical) o 1:1 (cuadrado)

El video se puede subir en la misma sección de media del producto.

⚠️ NO INCLUIR EN IMÁGENES
  • Logos de otras plataformas (Amazon, Shopify, etc.)
  • Texto promocional excesivo ("50% OFF!", "BEST SELLER!")
  • Watermarks o marcas de agua
  • Imágenes borrosas, pixeladas o con mal lighting
  • Claims médicos no aprobados

Configurar Variantes (Tallas, Colores)

3.1 — Activar Variantes

En la sección de producto, activa Product Variations.

Puedes crear hasta 2 niveles de variantes:

Nivel 1: Color, Sabor, Material, Estilo
Nivel 2: Talla, Tamaño, Cantidad, Pack
Ejemplo: Color (Black, White, Blue) × Talla (S, M, L, XL) = 12 variantes

3.2 — Configurar cada Variante

Para cada combinación de variantes, debes definir:

  • SKU único por variante
  • Precio (puede variar por variante)
  • Stock/Inventario individual
  • Imagen específica de la variante (ej: foto del color real)
💡 PRO TIP

Siempre sube una imagen por cada variante de color. Los listings sin fotos de variantes tienen tasas de retorno mucho más altas porque el cliente "no sabía cómo se veía" realmente.

Carga Masiva de Productos (Bulk Upload)

4.1 — Descargar Template

Navega a: Seller Center → Products → Batch Tools → Import Products

  1. Selecciona la categoría de los productos que vas a subir
  2. Clic en Download Template
  3. Abre el archivo Excel descargado
  4. Llena cada fila = 1 producto, siguiendo las columnas del template
  5. Las columnas en rojo son obligatorias

4.2 — Subir el Archivo

  1. Vuelve a Batch Tools → Import Products
  2. Clic en Upload File
  3. Selecciona el Excel completado
  4. Espera el procesamiento — TikTok valida cada fila
  5. Si hay errores, descarga el reporte de errores y corrige
  6. Las imágenes se deben subir por separado o usar URLs públicas

Categorías, Tags y SEO de Producto

6.1 — Cómo Funciona el SEO en TikTok Shop

TikTok Shop tiene su propio motor de búsqueda. Los compradores buscan productos dentro de la app. Tu producto aparece según:

  • Título del producto — Factor #1 de ranking
  • Categoría asignada — Match con la intención de búsqueda
  • Atributos del producto — Filtros que usan los compradores
  • Descripción — Keywords secundarias
  • Ventas y reviews — Social proof = mejor ranking
  • Shop Performance Score — Tiendas con mejor SPS rankean más alto

6.2 — Research de Keywords

No hay una herramienta oficial de keywords en TikTok Shop, pero puedes:

  1. Buscar tu producto en TikTok Shop y ver qué autocompletar sugiere
  2. Ver los títulos de los top sellers en tu categoría
  3. Revisar hashtags populares en videos de tu nicho
  4. Usar TikTok Ads Manager → Keyword Insights (si tienes acceso)
  5. Revisar keywords de Amazon/Google para el mismo producto
